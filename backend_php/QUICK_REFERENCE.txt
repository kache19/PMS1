## Pre-Deployment

1. **Change Password** ⚠️
   - cPanel → Email Accounts → Change Password
   - Update .env MAIL_PASSWORD

2. **Configure DNS**
   - Add SPF record
   - Enable DKIM
   - Configure DMARC
   - Wait 24-48 hours

3. **Test Locally**
   - Visit test_email_web.php
   - Send test email
   - Verify receipt

---

## Deployment

1. Upload modified files to production
2. Set permissions: `chmod 644 .env`
3. Run verification script
4. Send test email
5. Monitor error_log

## If Email Fails

### Immediate Fallback (Gmail)
```ini
MAIL_HOST=smtp.gmail.com
MAIL_PORT=587
MAIL_USERNAME=your.email@gmail.com
MAIL_PASSWORD=16_char_app_password
MAIL_ENCRYPTION=tls
```

### Temporary Testing (Mailtrap)
```ini
MAIL_HOST=live.smtp.mailtrap.io
MAIL_PORT=587
MAIL_USERNAME=mailtrap_username
MAIL_PASSWORD=mailtrap_password
MAIL_ENCRYPTION=tls
```
